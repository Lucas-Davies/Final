About

Jody Graham is a Sydney-based multidisciplinary artist working across drawing, mixed media, installation and performance, often using unconventional tools and found materials. Themes include the broken and discarded, regeneration after fire, and Australian birdlife. Series include Getting Pinked, Burnt, Fly like a Bird, Behind the Facade, and performance projects such as Sound–Drawing. Recent activity includes Burnt into Memory (Hawkesbury Regional Gallery, Jun–Aug 2025) and Drawn from Memory (Lost Bear Gallery, Aug 2025).
